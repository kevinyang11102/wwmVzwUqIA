# 人事管理系统简介 java372

## 项目概述

本项目为人事管理系统，基于Springboot+vue的前后端分离架构。采用MVC三层体系结构，结合Spring、SpringBoot、Mybatis、Vue及Maven技术栈，打造一套界面简洁、操作简便的Web系统。

## 技术栈

- 前端：Vue
- 后端：Springboot
- 数据库：MySQL
- 构建工具：Maven

## 功能模块

系统主要包含以下功能模块：

- 用户管理
- 员工信息管理
- 部门管理
- 职位管理
- 薪资管理

## 系统角色

- 系统管理员
- 普通用户

## 运行环境

- 建议使用现代浏览器，如Chrome、Firefox等。
- 后端环境：Java 1.8+
- 数据库：MySQL 5.7+

## 部署步骤

1. 安装MySQL数据库，导入数据库脚本。
2. 使用Maven构建项目，生成可执行jar包。
3. 部署至Java应用服务器，如Tomcat。
4. 前端项目使用Vue CLI启动。

## 目录结构

```
.
├── front-end/    # 前端项目目录
└── back-end/     # 后端项目目录
    ├── src/
    ├── pom.xml
    └── ...
```

## 核心亮点

- 前后端分离，便于管理。
- 使用Springboot和Vue，提升开发效率。
- 系统界面简洁，操作简单，易于上手。
- 基于MVC架构，模块化设计，方便扩展。
